Detailed Description

nRF SDK FDS operations and handlers for BLEAM Scanner.

BLEAM Scanner stores some of its data, like Configuration data, in flash permanently.

For logic behind flash data storage usage, please refer to BLEAM Scanner 2 wiki: Flash data storage.

Function for erasing all flash pages.

This function erases all flash pages in case they couldn't be initialised. It commonly occurs after a big DFU update, like from RuuviTag firmware to BLEAM Scanner bootloader + SD to BLEAM Scanner firmware.
