Detailed Description

Support of system time on a BLEAM Scanner node.

BLEAM Scanner supports synced system time for better maintenance. 

     For details, please refer to [BLEAM Scanner 2 wiki: System time](https://github.com/Connax-Oy/Bleam-Scanner-2/wiki/System-time).
