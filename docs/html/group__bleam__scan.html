Detailed Description

BLEAM Scanner's scanning for BLEAM.

BLEAM Scanner's key functionality is scanning for BLE packets from BLEAM devices. 

     For details, please refer to [BLEAM Scanner 2 wiki: Scan](https://github.com/Connax-Oy/Bleam-Scanner-2/wiki/Scan).
